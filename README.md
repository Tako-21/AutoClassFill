# 42 CPP Copilot

This extension improve your experience in the CPP Pool. It helps you to implement automatically canonical class in C++.

## Install
Launch Quick Open with Ctrl+P or ⌘+P and enter
`ext install bbird21.42cpp-copilot`

## Usage
Launch Quick Open with Ctrl+P or ⌘+P and enter
`cppCopilot`

**Enjoy!**
